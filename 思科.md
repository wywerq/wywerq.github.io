## 基本命令

| 功能     | 命令                 |
| -------- | -------------------- |
| 特权模式 | `enable`             |
| 配置终端 | `configure terminal` |
| 设备改名 | `hostname <设备名>`  |
| 路由功能 | `ip routing`         |
| 退出     | `exit`               |

## 接口

| 功能       | 命令                                                |
| ---------- | --------------------------------------------------- |
| 进入接口   | `interface <接口>`</br>`interface range <接口范围>` |
| 配置IP地址 | `ip address <IP地址> <子网掩码>`                    |
| 加入 VLAN  | `switchport access vlan <VLAN号>`                   |
| 启用接口   | `no shutdown`                                       |

### 端口

| 功能     | 命令            |
| -------- | --------------- |
| 交换端口 | `switchport`    |
| 路由端口 | `no switchport` |

### Trunk 接口

| 功能         | 命令                                   |
| ------------ | -------------------------------------- |
| 封装         | `switchport trunk encapsulation dot1q` |
| 配置         | `switchport mode trunk`                |
| 配置时钟频率 | `clock rate <频率>`                    |

### 配置虚拟子接口

1. 启用主接口 `interface <主接口>` `no shutdown`
2. 创建子接口 `interface <子接口>`
3. 封装子接口 `encapsulation dot1Q <VLAN号>`
4. 配置IP `ip address <IP地址> <子网掩码>`

## VLAN

| 功能 | 命令            |
| ---- | --------------- |
| 创建 | `vlan <VLAN号>` |
| 改名 | `name <VLAN名>` |

### 配置 VLAN IP

1. 进入VLAN `interface vlan <VLAN号>`
2. 配置IP `ip address <IP地址> <子网掩码>`

## 路由协议

网段地址：IP地址和子网掩码与运算

### RIP

| 功能     | 命令                 |
| -------- | -------------------- |
| 配置路由 | `route rip`          |
| 配置网络 | `network <直连网段>` |

### RIPV2

| 功能         | 命令                 |
| ------------ | -------------------- |
| 配置路由     | `route rip`          |
| 配置版本     | `version 2`          |
| 关闭自动汇总 | `no auto-summary`    |
| 配置网络     | `network <直连网段>` |

### OSPF

| 功能     | 命令                                        |
| -------- | ------------------------------------------- |
| 配置路由 | `route ospf <进程号>`                       |
| 配置网络 | `network <直连网段> <反掩码> area <区域号>` |

### 静态路由

    ip route <非直连网段> <子网掩码> <下一跳地址/端口>

### 缺省/默认路由

    ip route 0.0.0.0  0.0.0.0 <下一跳地址>

## PPP 协议

**两端封装：`encapsulation ppp`**  
封装完再配置认证

### PAP 认证

| 认证类型 | 命令                                                                                                                                                                      |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 单向认证 | **认证端配置：**<br>`ppp authentication pap`<br>`username <被认证端用户名> password <密码>`<br><br>**被认证端配置：**<br>`ppp pap sent-username <用户名> password <密码>` |
| 双向认证 | **两端配置：**<br>`ppp authentication pap`<br>`username <对方用户名> password <对方密码>`<br>`ppp pap sent-username <对方用户名> password <对方密码>`                     |

### CHAP 认证

| 认证类型 | 命令                                                                                                               |
| -------- | ------------------------------------------------------------------------------------------------------------------ |
| 单向认证 | **认证端配置：**<br>`ppp authentication chap`<br><br>**两端配置：**<br>`username <对方用户名> password <对方密码>` |
| 双向认证 | **两端配置：**<br>`ppp authentication chap`<br>`username <对方用户名> password <对方密码>`                         |
