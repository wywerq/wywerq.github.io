## CMD

| 功能        | 描述               |
| ----------- | ------------------ |
| 查看IP配置  | ipconfig /all      |
| 释放IP      | ipconfig /release  |
| 更新IP      | ipconfig /renew    |
| 清除DNS缓存 | ipconfig /flushdns |
| 解析域名    | nslookup <域名>    |

## DHCP

| 功能       | 描述                                      |
| ---------- | ----------------------------------------- |
| 安装       | 要求输入父域时，删除DNS进入下一步         |
| 新建作用域 | 范围默认1-254，再排除网关；必须配置网关IP |

## DNS

| 功能                   | 描述                                     |
| ---------------------- | ---------------------------------------- |
| 新建区域               | 新建完正向查找区域，必须新建反向查找区域 |
| 新建别名、邮件交换记录 | 主机名用www                              |
| 新建指针               | 主机名用www                              |

## IIS

| 功能    | 描述                          |
| ------- | ----------------------------- |
| 安装    | 角色服务根据需要选择FTP服务器 |
| 访问FTP | ftp:ip[:端口]               |
